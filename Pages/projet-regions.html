Description du projet

Ce projet consiste en le développement d'une application C# permettant d'explorer les regions de France. L'utilisateur peut naviguer entre les régions, consulter les départements qui les composent, puis accéder à la liste des villes de chaque département.

Fonctionnalités

  • Liste des 18 régions de France (métropole et outre-mer)
  • Affichage des départements par région sélectionnée
  • Liste des villes par département avec code postal, latitude et longitude
  • Localisation sur carte interactive via coordonnées GPS
  • Données récupérées depuis un fichier JSON

Difficultés rencontrées

La principale difficulté a été l'extraction et la manipulation des données depuis le fichier JSON, notamment pour les convertir en objets C# utilisables. Je n'avais jamais eu a faire cela auparavant, et j'ai dû apprendre à utiliser des bibliothèques de sérialisation JSON pour y parvenir.
